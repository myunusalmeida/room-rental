BACA INI SEBELUM MELANJUTKAN

Langkah-langkah menjalankan aplikasi
1. Buat database di mysql, beri nama room_rental
2. Import database yang ada di folder db
3. Jalankan aplikasi

Informasi folder
1. Config => Folder config berisi beberapa perintah PHP seperti pemanggilan database,
                aksi pada CRUD dan beberapa hal lainnya yang berhubungan data dan database.
2. Layouts => Folder layouts berisi potongan kode yang sangat penting pada aplikasi.
                Saya membagi kode ke beberapa bagian agar mempermudah development dan memakai
                metode include pada aplikasinya atau bisa kita sebut dengan puzzle.
                Potongan kode yang saya letakkan di layouts adalah kode-kode penting seperti
                header pada HTML, navigation bar, dan footer.
                Silahkan membuka folder dan membuka file tersebut agar sedikit paham.
3. Styles => Seperti namanya, disini saya meletakkan style atau CSS nya.
4. Vendors => Folder ini berisi framework dan beberapa kode dari luar seperti bootstrap.

Jika masih ada pertanyaan silahkan hubungi kami yaa.
